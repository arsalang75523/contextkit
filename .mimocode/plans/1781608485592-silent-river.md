# Project Summary — ContextKit (bankr-agent)

## In chie?

ContextKit yek platform zir-sakht-e context baraye AI agents-e khodmoharreke. Esme package npm-e `@basedchef/contextkit` va domaene-esh `contextkit.pro`.

Be zaban-e sade: yek servise SaaS ke mokalemat-e toolani va tarikhche-ye project ro be khorooji-haye foghshode va sakhtyafte tabdil mikone — baraye hafezeye agent, handoff beyn-e agent-ha, estekhraj-e profile-e karbar, va edame-ye kar.

## Chi kar mikone?

6 ta endpoint asli dare:

| Endpoint | Karborad | Gheymat |
|---|---|---|
| `POST /api/summarize` | Khales-sazi-e context (micro/compact/extended) | $0.05 |
| `POST /api/compress-context` | Foghshode-sazi-e context-e machine | $0.03 |
| `POST /api/handoff` | Enteghal-e project beyn-e agent-ha | $0.03 |
| `POST /api/extract-profile` | Estekhraj-e profile-e dadegi-e karbar | $0.04 |
| `POST /api/memory-enrichment` | Takhmol-e hafezeye boldmoad | $0.04 |
| `POST /api/tokens/estimate` | Takhmin-e token | API key |

## Me'mari (Architecture)

- **Framework:** Next.js 15 + Hono (API routes)
- **Database:** PostgreSQL ba Drizzle ORM
- **UI:** React 19 + Tailwind CSS + Framer Motion
- **Pardakht:** x402 (pardakhte crypto USDC rooye Base) + API key credits
- **LLM Backend:** Bankr LLM Gateway (`llm.bankr.bot`)
- **SDK:** package-ye TypeScript joda dar `packages/sdk/`
- **Deploy:** Docker (Hetzner) + Vercel + Cloudflare Pages

### Sakhtare-ye `src/`:

```
src/
├── app/              # Next.js pages (dashboard, playground, docs, pricing, ...)
├── routes/           # Hono API routes (auth, context, analytics, webhooks, tokens)
├── services/         # Business logic (account, credit, payment, context, analytics, ...)
├── prompts/          # LLM prompt templates
├── db/               # Drizzle ORM schema & migrations
├── components/       # React UI components
├── middleware/       # Auth & rate limiting middleware
├── webhooks/         # Webhook delivery system
├── lib/              # Shared utilities
├── types/            # TypeScript types
├── utils/            # Helper functions
├── storage/          # Storage abstractions
├── content/          # Static content
└── server.ts         # Standalone Hono server entry
```

## Se masir-e estefade (Three Usage Paths)

1. **Bankr-hosted x402** — Sadtarin masir. Faghat `bankr x402 call` bezanid, ba Bankr pardakht konid, JSON begirid. Bedoone API key ya SDK.
2. **API key credits** — Karbaran-e dashboard USDC mikharan va ba API key mostaghim estefade mikonan.
3. **TypeScript SDK** — Tosee-dehandegan-e pishrafte ba `@basedchef/contextkit` integrat mikonan.

## Dashboard va Monitoring

- Dashboard-e web: modiriyat-e API keys, credits, usage, payments, webhooks
- Analytics: overview, tokens, payments, usage
- Webhooks: rooydad-haye completion ro be endpoint-e delkhah push mikone
- OpenAPI docs dar `/docs` va `/api-reference`

## Khales

In project yek **API-based context management service** baraye AI agents hast ke ba model-e pardakhte crypto (x402) kar mikone va agent-ha mitoona mokalemat-e toolani-shono foghshode konan, beyn-e ham handoff bedan, profile-e karbar estekhraj konan, va hafezeye boldmoad dashte bashan.
