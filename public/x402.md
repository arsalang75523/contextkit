# ContextKit Bankr x402 Endpoints

ContextKit uses four Bankr-hosted x402 lanes. The caller approves USDC payment and receives ContextKit JSON; no ContextKit API key or SDK is required.

Base URL:

```txt
https://x402.bankr.bot/0xdace98cd605dd56b2edc66f0f4df3687f64fd824
```

## Core Context Operations

Service: `contextkit-core`
Price: `$0.03`

Select summarize, compress-context, handoff, extract-profile, or memory-enrichment with `endpoint`.

```bash
bankr x402 call https://x402.bankr.bot/0xdace98cd605dd56b2edc66f0f4df3687f64fd824/contextkit-core \
  -X POST \
  -d '{"endpoint":"summarize","messages":[{"role":"user","content":"Summarize this project state for the next AI agent."}],"mode":"compact"}'
```

## Compile, Push, Or Publish A Skill Repository

Service: `contextkit-experience-write`
Price: `$0.01`

Compile completed, useful work from any legitimate domain into a private tested `SKILL.md` draft:

```bash
bankr x402 call https://x402.bankr.bot/0xdace98cd605dd56b2edc66f0f4df3687f64fd824/contextkit-experience-write \
  -X POST \
  -d '{"mode":"skill-compile","messages":[{"role":"user","content":"Repair the Bankr x402 timeout without changing the response contract."},{"role":"assistant","content":"Compared origin and gateway latency, precomputed the long request, and verified HTTP 200."}]}'
```

Compilation returns a private `skillId`, validation report, and one-draft `publishToken`. Publish only after `validation.eligible=true` and explicit user approval:

Before publishing executable work, send a complete file array with `mode: "skill-validate"`, then repeat with `mode: "skill-push"`. Required roots are `SKILL.md`, `skill.json`, and `LICENSE`; public code bundles also require package/lock, config schema, `src/`, `tests/`, and `examples/`. The push response returns an immutable SHA-256 digest.

```bash
bankr x402 call https://x402.bankr.bot/0xdace98cd605dd56b2edc66f0f4df3687f64fd824/contextkit-experience-write \
  -X POST \
  -d '{"mode":"skill-repository-publish","skillId":"exp_REPLACE_ME","publishToken":"pub_REPLACE_ME","userApproved":true,"priceUsd":0.05}'
```

## Long Context

For long context operations, upload first and call `contextkit-core` with the returned `contextId`:

```bash
cat > long-context.txt <<'CONTEXTKIT_LONG_CONTEXT'
Paste the long conversation or document here.
CONTEXTKIT_LONG_CONTEXT

curl -X POST "https://contextkit.pro/api/context/upload-text?endpoint=summarize&mode=compact" \
  -H "Content-Type: text/plain" \
  --data-binary @long-context.txt

bankr x402 call https://x402.bankr.bot/0xdace98cd605dd56b2edc66f0f4df3687f64fd824/contextkit-core \
  -X POST \
  -d '{"endpoint":"summarize","contextId":"ctx_REPLACE_ME","mode":"compact"}'
```
